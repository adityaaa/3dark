## üîß **TECHNICAL SETUP**

### Video Optimization:
```
After generating videos:

1. Convert to web-friendly format:
   - Use: https://cloudconvert.com (free)
   - Convert to: MP4 (H.264 codec)
   - Resolution: 1080p max
   - Compression: Medium

2. Reduce file size:
   - Target: Under 5MB
   - Use: https://www.freeconvert.com/video-compressor
   - Maintain quality above 80%

3. Create backup formats:
   - MP4 for most browsers
   - WebM for Chrome/Firefox (smaller size)
```

### Upload to Vercel:
```
1. Save videos in /public/videos/ folder
2. Use naming convention:
   - hero-glow-transition.mp4
   - product-tiger-glow.mp4
   - lookbook-day-night.mp4
```
