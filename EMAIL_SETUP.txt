# EMAIL SETUP - RESEND

## Quick Setup (5 minutes)

### Step 1: Sign Up for Resend
1. Go to: https://resend.com/signup
2. Sign up with your email: **3darkclothings@gmail.com** (or any email)
3. Verify your email

### Step 2: Get API Key
1. After login, go to: **API Keys** (in dashboard)
2. Click **Create API Key**
3. Name it: `3Dark Production`
4. Copy the API key (starts with `re_...`)

### Step 3: Add Domain (Optional but Recommended)
1. Go to **Domains** section
2. Click **Add Domain**
3. Enter: `3dark.in`
4. Follow DNS setup instructions (Vercel should handle this automatically)
5. Or use: `onboarding@resend.dev` for testing (no domain setup needed)

### Step 4: Add to Vercel
1. Go to: https://vercel.com/dashboard
2. Select your project
3. Go to: **Settings** → **Environment Variables**
4. Add these variables for **Production**:

```
RESEND_API_KEY=re_your_api_key_here
EMAIL_FROM=3Dark <orders@3dark.in>
```

Or if using test domain:
```
RESEND_API_KEY=re_your_api_key_here
EMAIL_FROM=3Dark <onboarding@resend.dev>
```

5. Click **Save**
6. Go to **Deployments** → Redeploy latest

### Step 5: Test Email
After deployment:
1. Place a test order on https://3dark.in
2. Check if you receive order confirmation email
3. Check Vercel logs for any errors

## Troubleshooting

### Email not received?
1. Check Vercel logs for errors
2. Verify RESEND_API_KEY is set in Vercel
3. Check spam folder
4. Verify domain is verified in Resend (if using custom domain)
5. Try using `onboarding@resend.dev` first (no domain setup needed)

### "API key not configured" error?
- RESEND_API_KEY not set in Vercel environment variables
- Redeploy after adding environment variable
